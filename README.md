# Proyecto-1-Limpieza-y-preparaci-n-de-datos-de-clientes-para-estrategia-de-fidelizaci-n
Proyecto 1: Limpieza y preparación de datos de clientes para estrategia de fidelización
En este proyecto trabajé con datos de clientes de una empresa de comercio electrónico con el objetivo de preparar la información para el desarrollo de un programa de fidelización y futuras estrategias de marketing.

La primera fase consistió en la limpieza y transformación de los datos, los cuales se encontraban en formato de listas de Python con inconsistencias en nombres, formatos y estructuras. Se estandarizaron los nombres de usuarios, se corrigieron formatos de edad y se validó la integridad de los registros para garantizar la calidad del dataset.

Además, se procesaron las listas de categorías favoritas y gastos asociados para calcular el gasto total por cliente, generando así una métrica clave para el análisis de comportamiento de compra. También se identificaron y corrigieron posibles errores en los datos, asegurando coherencia entre las categorías y los valores de gasto.

Como resultado, se obtuvo un conjunto de datos limpio, estructurado y listo para análisis posteriores, enfocado en la segmentación de clientes basada en variables demográficas y de consumo.

Este proyecto demuestra habilidades fundamentales en limpieza de datos, transformación de estructuras, validación de información y preparación de datasets para análisis, sentando las bases para la generación de indicadores clave de negocio (KPIs) y la toma de decisiones basada en datos.
